How to Create an Account

  1. Click "Add Accounts."
  2. Select the type of account you want to create.
  3. An account of the selected type will be created.
  4. A screen will appear indicating the new account's ID number, owner name, type, and balance. If it is a checking account, it also shows the overdraft fee that will be incurred if the account balance becomes negative. If it is a savings account, it will show the minimum required balance, current monthly withdrawals (0), and max monthly withdrawals.
